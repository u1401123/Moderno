1. создайте новый репозиторий в командной строке

Создание локального репозитория

git init

Затем с помощью + добавить файлы в репозиторий (git add . добавляет все файлы)

Добавить контрольную точку

git commit -m "комментарий"
 
Создать ветку проекта

git branch -M main

Добавить репозиторий

git remote add origin https://github.com/u1401123/название.git

Отправить выбранные файлы в репозиторий github

git push -u origin main

2. Загрузить репозиторий

Копируем ссылку на репозиторий

https://github.com/u1401123/Moderno.git

git clone https://github.com/u1401123/Moderno.git

Или

git pull

обновлем локальный репозиторий содержимым с GitHub.

После изменения файлов нажимаем + тем самым сохраняем изменения.
Затем нажимаем галочку Commit, пишем комментарий и сохраняем контрольную точку.

Или 

git commit -m "комментарий"

Затем

git push

залить все на GitHub.

3. Алиасы или короткие команды записываются в gitconfig по адресу C:\Users\Игорь\.gitconfig

4. Откат к предыдущиму состоянию проекта на локальном репозитории.

git checkout -- index.html (git checkout . - удаляет изменения во всех файлах)

файл в котором отменить изменения.

Если изменения уже сохранены на локальном репозитории, то

git reset index.html

затем 

git checkout -- index.html

Если изменения уже закомментированы

git reset --hard HEAD^1

вернуться на 1 commit назад.
